# Air Plinko — Festival Booth Build Plan

## Concept
Hybrid plinko / skee-ball / air hockey booth game. Two players launch whiffle balls into a tilted, enclosed playing field and use aimed air nozzles to guide balls into scoring ledges. Inspired by the handheld waterful ring toss toy.

## Modes
- **Competitive:** Red ball vs Blue ball — highest score wins
- **Cooperative:** Same color — combine scores, beat a target

---

## Overall Dimensions (Recommended)

| Dimension | Size | Notes |
|-----------|------|-------|
| Width | 48" (4 ft) | Two-player side-by-side |
| Height | 60" (5 ft) | Playable standing for kids & adults |
| Depth (gap) | 5" | Whiffle ball (2.75") + airflow clearance |
| Tilt angle | ~15–20° from vertical | Keeps balls in play, gravity does the return work |

Total footprint with easel/frame legs: ~48"W × 36"D × 60"H

## Architecture

### 1. Frame & Playing Field
- **Back panel:** 4×5 ft sheet of ¾" plywood (painted matte — reduces friction)
- **Front panel:** 4×5 ft clear acrylic sheet (¼" thick — rigid, shatter-resistant)
- **Spacers/pegs:** ¾" wooden dowels (or PVC pipe segments), ~3" long, drilled through acrylic into plywood. Serve dual purpose:
  - Structural spacers holding acrylic at consistent 5" gap
  - Plinko pegs that create chaotic ball movement
- **Edge rails:** 1×6 lumber on all four sides, sandwiched between plywood and acrylic, forming the sealed playing field
- **Tilt frame:** A-frame easel legs on the back, adjustable angle with a chain or strap

### 2. Launcher System (×2)
- **Type:** Spring-loaded pinball plunger
- **Position:** Bottom-left and bottom-right of the playing field
- **Launch channel:** Short angled tube/ramp that feeds the ball up and into the playing field
- **Parts:**
  - Pinball plunger assemblies (~$8 each online)
  - 1.5" PVC pipe for launch channel
  - Ball return feeds directly into the launch chamber

### 3. Air System
- **Blower:** Single electric blower (bouncy house style inflator, ~$30–40, 2.0 PSI / 450W)
- **Distribution:** 2" flex hose from blower, Y-split to two player stations
- **Nozzles:** ¾"–1" rigid pipe/tube, ~6" long
- **Pivot mount:** Each nozzle mounted on a pivot bolt through the bottom edge rail, with stop bolts limiting to ~30° sweep
- **Nozzle position:** Bottom-center area of each player's half, aimed upward into the field
- **Flow control:** Consider a simple gate valve on each line for balance tuning

### 4. Scoring Ledges
- **Count:** 4 ledges at varying positions
- **Construction:** Small shelf brackets or angled wood strips mounted to back panel, with a lip to catch balls
- **Suggested layout & scoring:**

```
        [100 pts] ← top center, smallest, hardest to reach
       /          \
   [50 pts]    [50 pts] ← upper sides
      |            |
      |            |
   [25 pts]    [25 pts] ← mid sides, larger
      |            |
   [10 pts ———————— 10 pts] ← wide bottom shelf, easiest
```

- **Ledge depth:** ~3" — enough to catch and hold a whiffle ball
- **Lip height:** 1" — ball rolls in but doesn't roll out easily
- **Drainage holes (optional):** Small holes in ledge backs so balls eventually roll through to a collection chute behind the board, keeping the game flowing

### 5. Ball Return
- **Gravity-fed:** Balls that miss all ledges fall to the bottom of the playing field
- **Collection trough:** Angled channel at the bottom funnels balls back to each player's launcher
- **Divider:** Center divider in the trough sends balls to the correct player's side
- **From scoring ledges:** If using drainage holes, balls drop behind the board into a rear chute that feeds the same bottom return

## Build Sequence

### Phase 1: Frame & Board (Day 1)
1. Cut plywood to 48×60"
2. Cut edge rails from 1×6, attach to plywood edges with screws (forms the box)
3. Drill peg holes in plywood — grid pattern, ~6" spacing, offset rows (plinko style)
4. Insert and glue dowel pegs (3" long, protruding into playing field)
5. Build A-frame legs, attach with hinges

### Phase 2: Scoring & Return (Day 2)
1. Build and mount 4 scoring ledges inside the box
2. Build bottom collection trough with center divider
3. Build launch channels (PVC) at bottom corners
4. Install pinball plungers
5. Test ball flow: launch → field → ledge or return → back to launcher

### Phase 3: Air System (Day 3)
1. Mount nozzle pivots on bottom edge rail
2. Run flex hose from blower through Y-split to nozzles
3. Install rotation stops (30° range)
4. Power on and tune — adjust nozzle diameter or gate valves until air moves the whiffle ball effectively at various field positions
5. Test with multiple balls in play simultaneously

### Phase 4: Seal & Finish (Day 4)
1. Paint the back board (dark matte for visibility)
2. Mark scoring zones with paint/vinyl
3. Drill acrylic for peg pass-throughs (match plywood holes)
4. Mount acrylic front panel onto pegs/edge rails
5. Seal all edges — balls should not escape
6. Final playtesting
