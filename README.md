# Air Plinko

Hybrid plinko / skee-ball / air hockey festival booth game for two players.

Players launch whiffle balls into a tilted, peg-filled playing field, then use pivoting air nozzles to guide their ball into scoring ledges. Think: the handheld waterful ring toss game, supersized.

## Modes
- **Competitive:** Red ball vs Blue ball — highest score wins
- **Cooperative:** Same color — beat a target score together
